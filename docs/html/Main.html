Main

Description

Main module for the Gloss implementation of the game. It handles the window creation, redirects the drawing and reaction to events to the correct modules, and loads the PictureSet used throughout the program's lifespan.

Usage of pictures

Pictures were used, many times in place of text, due to the limitations of gloss when it comes to text rendering, namely custom fonts.
