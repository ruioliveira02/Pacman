type Coords = (Int, Int) #

Coordinates in the maze, given as a (y, x) pair The maze can be as a two-dimensional grid, where the origin (0,0) is the top left corner. The x-axis is the horizontal axis and the y-axis the vertical axis. The y coordinate increases as you go down, and the x coordinate increases as you go to the right

data Play #

Represents an action made by a player Each action is a movement of a player in a certain direction
