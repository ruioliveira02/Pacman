data Play #

Represents an action made by a player Each action is a movement of a player in a certain direction

type Coords = (Int, Int) #

Coordinates in the maze The maze can be as a two-dimensional grid, where the origin (0,0) is the top left corner. The x-axis is the horizontal axis and the y-axis the vertical axis. The y coordinate increases as you go down, and the x coordinate increases as you go to the right

compareCoords #

Arguments

:: Coords

The first set of coordinates

-> Coords

The second set of coordinates

-> Bool

Whether they are equal or not

Tests wether two coordinates are the same Eq is a class, only applies to data types. Coords is a tuple

Examples

>>> compareCoords (1,2) (1,2)
True

>>> compareCoords (1,3) (3,1)
False
